HR Configuration Wizard

The configuration wizard has been designed to aid the user in setting up the application in the right manner and to avoid the confusion regarding how to proceed further after a certain step.

It consists of 3 steps:

  1. Configure Leave Types: Create leave types (E.g. Sick, Vacation) and allocate the number of days for each leave type.

  1. Configure Holidays: Create holiday groups and assign holidays to each group.

  1. Appraisals: Add appraisal parameters (E.g. KPI, KRA) and allot questions for each parameter.

Create New Shortcut Button

This shortcut button is placed on the top left corner (beside your organization logo) of the application. This button has been designed to save your time from menu redirections. The number of options displayed depends on a user’s role privileges. For example, if you belong to the Management role group, then you will be able to create new Employees, Requisitions, Appraisals, Business Units, and Departments etc. If you belong to the Employee role group you can only create Service and Leave requests.

Contacts

You can now manage and store your organization’s external users, vendors, and clients’ details all in one place.

  1. Click HR in the top menu
  2. Click Contacts on the left menu panel
  3. You can click on any contact to add/view External Users/Vendors/Clients.
