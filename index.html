Sentrifugo User Guidance

This is a post installation guide to provide users information on how to use Sentrifugo easily. If you require instructions for installing Sentrifugo, please follow this link: http://www.sentrifugo.com/installation-guide. This guide comprises of a detailed description of Sentrifugo’s features, capabilities, and step-by-step procedures to use this application efficiently.

Prerequisite Skills

Sentrifugo end users do not require any specialized or additional technical skills to use the application.

How to Use This Guide

We would recommend you to follow this guide in a sequential order of the chapters. The chapters have been placed in a progressive order to help you to use Sentrifugo optimally. Sentrifugo 3.0 User Guide’s chapters consist of ‘How to’ questions to make it easier for users to locate solutions for their queries.
